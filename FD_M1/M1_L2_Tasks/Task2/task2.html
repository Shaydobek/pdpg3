Frontend dasturlash texnologiyalari
   

HTML - frontend yo'nalishining asosiy texnologiyasi bo'lib, unda asosan interfeysning asosi quriladi.
   

CSS - frontend yo'nalishining interfeys dizayniga javob beradigan asosiy texnologiyasi.
   

JavaScript - frontend yo'nalishining interfeysni jonlashtirish uchun mo'ljallangan daturlash tili.
